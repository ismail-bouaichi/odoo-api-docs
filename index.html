🔐 Authentification

Tous les endpoints (sauf /health) nécessitent une clé API. Vous pouvez la fournir de deux manières :

Option 1 — En-tête HTTP (recommandé)

X-API-Key: votre_clé_api

Option 2 — Paramètre de requête

GET /api/v1/properties?api_key=votre_clé_api
⚠️
Sécurité : Avec les paramètres de requête, la clé API peut apparaître dans les logs serveur et l'historique du navigateur. Utilisez la méthode par en-tête pour les intégrations en production.

🚀 Exemples

cURL

# Vérification de santé
curl https://admin.reliefresidences.ma/api/v1/health

# Lister toutes les propriétés résidentielles disponibles à la vente
curl -H "X-API-Key: VOTRE_CLÉ" \
  "https://admin.reliefresidences.ma/api/v1/properties?type=residential&stage=available&sale_lease=for_sale"

# Obtenir les F3 entre 500K et 800K MAD
curl -H "X-API-Key: VOTRE_CLÉ" \
  "https://admin.reliefresidences.ma/api/v1/properties?unit_morphology=f3&min_price=500000&max_price=800000"

# Obtenir toutes les unités du projet ID 2
curl -H "X-API-Key: VOTRE_CLÉ" \
  "https://admin.reliefresidences.ma/api/v1/properties?project_id=2"

# Obtenir les détails d'un projet avec ses blocs
curl -H "X-API-Key: VOTRE_CLÉ" \
  "https://admin.reliefresidences.ma/api/v1/projects/2"

# Pagination : page 3, 20 éléments par page
curl -H "X-API-Key: VOTRE_CLÉ" \
  "https://admin.reliefresidences.ma/api/v1/properties?page=3&per_page=20"

Python

# pip install requests
import requests

BASE_URL = "https://admin.reliefresidences.ma/api/v1"
HEADERS = {"X-API-Key": "VOTRE_CLÉ"}

# Lister les propriétés disponibles
response = requests.get(
    f"{BASE_URL}/properties",
    headers=HEADERS,
    params={
        "stage": "available",
        "type": "residential",
        "per_page": 100,
    }
)
data = response.json()

for prop in data["data"]:
    print(f"{prop['code']} — {prop['name']} — {prop['price']} {prop['currency']}")

print(f"Total : {data['pagination']['total']} propriétés")

JavaScript / Node.js

// Utilisation de fetch
const BASE_URL = "https://admin.reliefresidences.ma/api/v1";
const API_KEY = "VOTRE_CLÉ";

async function getProperties(filters = {}) {
  const params = new URLSearchParams(filters);
  const response = await fetch(
    `${BASE_URL}/properties?${params}`,
    { headers: { "X-API-Key": API_KEY } }
  );
  return response.json();
}

// Obtenir tous les F2 réservés
const result = await getProperties({
  stage: "booked",
  unit_morphology: "f2",
});

console.log(`Trouvé ${result.pagination.total} propriétés`);
result.data.forEach(p =>
  console.log(`${p.code} — ${p.name} — ${p.price} ${p.currency}`)
);
